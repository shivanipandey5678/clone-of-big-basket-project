# Big Basket Clone

Welcome to the Big Basket Clone project repository! This project aims to replicate the functionality of the Big Basket website, providing users with a platform to shop for groceries online. The project features a responsive design, navigation bar for easy page access, product sorting options, and a search functionality tailored for fruits and vegetables.

## Features

- **Navigation Bar:** Easily navigate between different pages of the website using the responsive navigation bar located at the top.
- **Responsive Design:** The website is designed to adapt to various screen sizes, ensuring a seamless user experience across devices.
- **Product Sorting:**
  - Sort products based on:
    - Rate
    - Discount
    - Increasing order of price
    - Decreasing order of price
  - Filter products based on price range and select options such as above, below, or equal to the specified price.
- **Search Functionality:**
  - Search for specific fruits and vegetables by name on the Exotic Fruits and Vegetables page.

## Usage

To run the project locally, follow these steps:

1. Clone this repository to your local machine using `git clone`.
2. Navigate to the project directory.
3. Open the `index.html` file in your web browser.
4. Explore different pages using the navigation bar.
5. Try out the product sorting and search functionalities.
